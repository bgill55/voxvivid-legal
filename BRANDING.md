# VoxVivid Branding 💎🌊

Welcome to the VoxVivid Brand Kit. This directory acts as the source of truth for our visual identity.

## 📂 Asset Structure

All high-fidelity assets are located in `/public/branding`:

- **`/icons`**: High-resolution "Mirror Identity" logos.
- **`/social`**: Cinematic social preview images (OG images).
- **`/fonts`**: Reference links to our typography.

## 🎨 Visual Identity

| Element | Specification |
|---------|---------------|
| **Core Accent** | `#9d00ff` (Purple) |
| **Secondary Accent** | `#00f5ff` (Cyan) |
| **Background** | `#0a0a0a` (Obsidian) |
| **Typography** | Space Grotesk (Headings), Inter (Body) |
